---
title: TypeScript API with Bun
---

A backend API built with Bun to learn server-side development, schema
validation, and data persistence.

[GitHub Repository - Bun API](https://github.com/nekrosb/bun-todo-api-nekrosb)

## Technologies involved

- **TypeScript**: Written entirely in TypeScript for robust backend types.
- **Bun**: Fast JavaScript runtime and built-in server capabilities.
- **Valibot**: For schema validation and ensuring data integrity.
- **SQLite**: Local database to reliably store the data.
