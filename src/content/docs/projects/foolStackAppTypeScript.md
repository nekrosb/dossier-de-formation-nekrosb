---
title: TypeScript Full Stack Application
---

A full-stack web application built entirely with TypeScript, with a modern
frontend, a lightweight backend, and PostgreSQL for data storage.

## Technologies involved

- **TypeScript**: Used across both the frontend and backend to ensure type safety and maintainable code.
- **React**: Built the frontend using reusable components and a modern development workflow.
- **Elysia**: Powered the backend with a lightweight and high-performance TypeScript framework.
- **Bun**: Served as the JavaScript runtime and package manager for fast development and execution.
- **PostgreSQL**: Used as the relational database for storing and managing application data.
