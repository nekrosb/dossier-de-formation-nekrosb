---
title: PHP Blog
---

A blog application built with pure PHP and SQLite to practice backend
development without frameworks.

## Technologies involved

- **PHP**: Used as the core backend language to handle routing, business logic, and server-side rendering.
- **SQLite**: Provides a lightweight database for storing blog posts and application data.
- **HTML & CSS**: Used to create the user interface and structure the application.
- **Pure PHP**: The project was developed without frameworks, focusing on core PHP concepts, database interactions, and CRUD operations.
